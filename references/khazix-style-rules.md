# 卡兹克公众号写作风格 — 可程序化复用规则手册

> 一句话定义：「有见识的普通人在认真聊一件打动他的事。」

---

## 一、5个核心风格锚点

### 锚点1：活人感优先（Anti-AI Detection）
- 所有输出必须听起来像真人在跟朋友聊天，不像在写报告
- 判断标准：读出来不尴尬，能想象一个人坐在你对面说这段话
- 实现手段：口语化词组高频使用、故意打破语法完整性、自嘲、情绪标点（。。。 ？？？ = =）
- 反面：任何读起来像「总结陈词」「学术论文」「品牌文案」的段落都需要重写

### 锚点2：亲自下场（First-Person Experience）
- 核心基因：不是隔空评论，是「我替你去做了这件事」
- 所有案例必须是正在发生的真实细节（「就像我今天正在搞的xxx」），禁止假设性例子（「比如有一次」）
- 工具名必须具体（Claude Code、Seedance 2.0），禁止空泛表述（「AI工具」「某个模型」）

### 锚点3：波动式节奏（Rhythm Wave）
- 长短句交替，一句话独立成段制造重点（全文至少3次）
- 围绕主线偏出去一点点（案例、知识、类比），然后用「扣主线句」拉回来
- 疑问句做「刹车和转向」，让读者停一秒准备接收新信息
- 禁止连续3句以上句式长度相近

### 锚点4：知识随手掏（Casual Knowledge Drop）
- 知识呈现方式是「聊着聊着自然想到了」，不是「下面我来科普」
- 触发句式：「突然想起了以前xxx的时候」「这玩意叫xxx」「想想当年xxx」
- 引用（论文、书籍、历史）必须像「聊天时脑子里冒出来的」，不能像「特意查的」

### 锚点5：文化升维（Cultural Elevation）
- 每篇文章在聊完具体事情后，连接到一个更大的文化/哲学/历史参照物
- 不是硬凑升华，是「聊着聊着自然想到了」的感觉
- 经典案例：9.9元DeepSeek→北京折叠、AI看不到爱心→格式塔心理学→「我们活在流中而AI活在帧中」

## 二、标题公式

卡兹克标题的底层逻辑是 **「具体事件 + 情绪钩子 + 信息差暗示」**

### 公式模板

| 公式 | 结构 | 示例 |
|------|------|------|
| 亲历体 | 我+动作+对象+转折发现 | 淘宝上卖9块9的DeepSeek，让我看到了被折叠的魔幻世界 |
| 数据体 | 分析了X个Y后+反直觉发现 | 分析了1.8亿个岗位后，我发现应届生们好像被AI堵在了门外 |
| 荒诞体 | 荒诞事实+情绪词 | 我花499找人上门安装OpenClaw |
| 热点体 | 火爆描述+解密承诺 | 周杰伦发的1400万人点赞的AI视频，是怎么做出来的？ |
| 反转体 | 常规认知+打破 | 假扮AI的17岁男高中生，用回复治愈了整个B站 |

### 标题规则
- 必须包含具体的数字、人名、产品名（不能抽象）
- 禁止「在AI时代xxx」「关于xxx你需要知道的」这类套话标题
- 标题本身要让人产生「然后呢？」的冲动
- 通过HKR质检：H(有趣/悬念) + K(信息量) + R(情绪共鸣)，至少占两项

## 三、开头模式（4种必杀技）

### 模式A：叙事启动
```
故事是这样的。
[一句话交代起因，制造好奇]
```
示例：「故事是这样的。我最近刷淘宝挺多，然后昨天，鬼使神差的，在淘宝上，搜了一下DeepSeek。」

### 模式B：荒诞事实
```
[直接抛出一个让人「？？？」的事实]
[补一句加深荒诞感]
```
示例：「我可能，刚刚成为了哈基米的儿子。至少，AI是这么认为的。」

### 模式C：热点破题
```
[最近/这两天 + 被xxx刷屏了/火了]
[补充火爆的具体证据]
```
示例：「Seedance2.0的火，已经烧了好几天了。抖音b站到处都是二创视频，微博科技榜一下子上了四个热搜。」

### 模式D：好奇心驱动
```
[最近刷到/看到/发现了一个xxx]
[很有意思/非常魔幻/让我xxx]
```
示例：「这两天在网上刷到了一张图，很有意思。」

### 开头绝对禁区
- 禁止「在当今AI快速发展的时代」
- 禁止「随着技术的不断进步」
- 禁止任何宏大叙事开场
- 第一句话必须让读者产生「然后呢？」

## 四、结尾模式（5种收法）

### 收法A：引用收尾
借别人的话收束，但用自己的节奏重新排列。
```
比比说：
我没什么其他的力量。
磨平一些信息差。
这好像，就是我为数不多能做的。
最棒的使命。
```

### 收法B：哲思余韵
一个短句留白，让读者自己回味。
```
还有，那时间。
流逝的本身。
```

### 收法C：行动呼吁
鼓励读者去做一件具体的事。
```
多花一点点时间，自己去研究一下。
这个过程，可能是AI时代。
比学会使用一个工具更重要的事。
```

### 收法D：信念宣言
表达对未来的坚信，但不是鸡汤，是经历过挣扎后的坚定。
```
但是我还是始终坚信。
这些墙在汹涌向前的洪流之下，必然会倒塌。
新时代，一定会到来的。
```

### 收法E：回环呼应
回到开头的意象，但视角已经不同（契诃夫之枪原则）。
```
能做的，还是那句话。
磨平一点点信息差。
哪怕，只是很小很小的一点。
那我也觉得，我做了一些正确的事。
```

### 固定尾部（每篇必加）
```
以上，既然看到这里了，如果觉得不错，随手点个赞、在看、转发三连吧，如果想第一时间收到推送，也可以给我个星标⭐～
谢谢你看我的文章，我们，下次再见。
> / 作者：卡兹克
> / 投稿或爆料，请联系邮箱：wzglyay@virxact.com
```

## 五、禁忌清单（硬性规则，零容忍）

### A. 禁用词（出现即替换）

| 禁用词 | 替换为 |
|--------|--------|
| 说白了 | 坦率的讲、其实就是 |
| 意味着什么 / 这意味着 | 那结果会怎样呢、所以呢 |
| 本质上 | 说到底、其实 |
| 换句话说 | 你想想看、也就是说 |
| 不可否认 | 直接删掉，换成正面陈述 |
| 综上所述 / 总的来说 | 用具体的回扣句 |
| 首先...其次...最后 | 用自然转场词替代 |
| 值得注意的是 / 不难发现 | 直接删掉，直接说 |
| 让我们来看看 / 接下来让我们 | 直接删掉 |

### B. 禁用标点（出现即替换）

| 禁用 | 替换为 |
|------|--------|
| 冒号「：」 | 用逗号「，」代替 |
| 破折号「——」 | 用逗号或句号代替 |
| 双引号「""」和「""」 | 用「」替代，或直接不加引号 |

### C. 禁用结构

- 禁止bullet point罗列观点（超过3个必须改成散文叙述）
- 禁止大段加粗（超过2行的加粗 = 过度结构化）
- 禁止使用小标题切割文章（除非是「N条心得」分条目结构）
- 板块衔接必须用口语化转场句（「说到这个」「回到xxx这块」）

### D. 禁用开头模式

- 「在当今xxx的时代」
- 「随着xxx的发展/进步」
- 任何宏大叙事开场

### E. 禁用内容模式

- 假设性例子：「比如有一次...」（要用「就像我今天正在搞的xxx」）
- 空泛工具名：「AI工具」「某个模型」（必须说具体名字）
- 教科书式科普：「下面我来介绍」「首先需要了解」
- 知识性情绪描述：「我感到非常震撼」（要用体感记忆：「我当时就愣住了」）

## 附录：推荐口语化词组库（全文至少使用8-10个不同的）

**转场**：坦率的讲、说真的、我是真的觉得、反正我觉得、怎么说呢、其实吧、你想想看、我跟你说、回到xxx这块、顺着上面的再聊聊

**判断**：我有时候觉得、我一直觉得、这话听着有点刺耳但、我自己的感受是、我始终坚信

**自嘲**：说实话我也不确定、我自己也还在摸索、可能有些想法还不成熟、愚钝如我

**情绪**：这种感觉太爽了、我当时就愣住了、太离谱了、你敢信？？？、给我一下子整不会了

**拉近**：很多朋友可能不知道、可能有小伙伴纳闷、大家也都知道

**口癖**：这玩意、不是哥们、有个屁的xxx、太牛逼了、比较骚的事

## 附录：四层自检体系（写完必跑）

L1 硬性规则：禁用词扫描 + 禁用标点扫描 + 结构套话扫描 + 工具名检查 → 零命中才通过
L2 风格一致性：开头检查 + 节奏检查 + 口语化检查（8-10个词组） + 标点禁令二次确认
L3 内容质量：观点支撑 + 知识输出方式 + 文化升维 + 对立面同理心 + 类型专项
L4 活人感终审：温度感（体感vs知识性） + 独特性 + 姿态（朋友vs导师） + 心流连续性
