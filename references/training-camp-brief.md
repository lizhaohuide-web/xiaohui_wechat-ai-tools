## 产品定位

帮月营收 5-50 万的小团队老板搭建 4 个能自主跑业务的 AI Agent。
4 周线上陪跑，陪你做出来，做不到全额退款。

## 四大 AI Agent

| Agent | 作用 | 对标人力 |
|-------|------|---------|
| 知识库 Agent | 组织大脑，回答内部问题 | 行政/文档管理 |
| 获客 Agent | 自动生产内容、投放获客 | 运营/市场 |
| 销售 Agent | 自动培育线索、推动转化 | 销售助理 |
| 交付 Agent | 自动客服、售后跟进 | 客服 |

## 核心差异化

1. **陪跑交付结果**：不是卖课，是陪你做出来。4 周结业有硬指标，达不到全退
2. **全栈兼容**：Hermes/Coze/Dify/n8n 按场景选，不绑定平台
3. **真实案例背书**：流金享老（养老行业）、育儿生图（内容行业）都是跑通的

- 月营收 5-50 万小团队老板，1-10 人
- 28-42 岁，一二线城市
- 用过 AI 但不系统，想省人力成本
- 行业：知识付费、咨询、本地生活、新媒体代运营、电商

```

```

## 红线（绝不能碰）

- ❌ 承诺收入：「学了就能月入 X 万」
- ❌ 贬低竞品：「比 XXX 好 N 倍」
- ❌ 虚假数据：编造案例/效果
- ❌ 限时逼单：「今天不买明天涨价」
- ❌ 文中间硬广：破坏阅读体验
